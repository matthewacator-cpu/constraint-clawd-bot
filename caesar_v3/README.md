# CAESAR v3.0 - Thermodynamic Hallucination Firewall

Production-ready AI hallucination detection with real API integration.

## Quick Start

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Add API key:**
   ```bash
   cp .env.example .env
   # Edit .env and add your OPENAI_API_KEY
   ```

3. **Run:**
   ```bash
   python ice_bridge.py
   ```

## Docker Deployment

```bash
docker-compose up -d
```

## What It Does

CAESAR prevents AI hallucinations by:

- Adaptive verification (Ice Protocol)
- Real confidence from logprobs
- Energy-based cost tracking
- Automatic blocking of uncertain outputs
